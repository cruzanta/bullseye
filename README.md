# Bullseye

A simple 2D dart game made with OpenGL and GLUT.

## Prerequisites

 * Git
 * A C++ compiler
 * OpenGL
 * GLUT
 * CMake

## Installation

To download, build, install, and run:

    git clone https://github.com/cruzanta/bullseye.git
    cd bullseye
    cmake .
    make
    ./bin/Bullseye

## Controls

| Key   | Action          |
| ----- | --------------- |
| space | Start power bar |
| s     | Stop power bar  |
| t     | Throw dart      |
| q     | Quit            |
